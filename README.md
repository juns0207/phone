## 안녕하세요 스마트폰 가이드입니다.

나에게 맞는 스마트폰을 추천하는 흥미진진 챗봇 아이오입니다.

저희는 자신의 연령대별, 자신이 원하는 가격으로 스마트폰을 추천해드립니다.

현재는 삼성폰과 애플폰만 지원합니다~.

저희 스마트폰 가이드는 스마트폰 추천은 물론이며, 스마트폰 중독 예방과 진단을 해드리고 있습니다!

챗봇을 통해 스마트폰 진단을 받으실 수 있으며 스마트폰 중독으로 진단된 분들을 위해 도움이 되는 사이트를 소개해 드리고 있습니다.
